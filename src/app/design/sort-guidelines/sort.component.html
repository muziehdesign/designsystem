Sort

To add sorting behavior to table headers, add mzSort directive to mz-result-table component and also add the mzSortKey directive to each table header and provide an id that will identify it.

Users can trigger the sort event through a mouse click. When this happens, the output (sort) will emit a sort-event that contains the ID of the header triggered

Example

            <mz-result-table [mzSort]="sort" (sort)="onSort($event)">
          
              <th mzSortKey="orderNumber">Order</th>
          
            </mz-result-table>

Best practices

  • Default sort is defined in the mzSort directive.
