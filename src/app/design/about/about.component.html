About

This is a work-in-progress design system that will allow quick bootstrap of modern new projects meeting the following needs
  • components
  • patterns
  • styles
  • accessibility
  • responsiveness

It adapts Atlassian Design System, and uses Tailwindcss to create css components and base styles.

About design system

A design system provides a unified design language by defining a set of standards on styles, patterns, and components. Each decision in this system should be driven by need, purpose, and reason. Having a unified design language alleviates the teams from redesign, promotes consistency, and reusability.
