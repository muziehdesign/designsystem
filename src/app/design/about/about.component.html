About

Muzieh Design System is a work-in-progress design system that will allow bootstrapping modern new projects quickly. Css styles are intentionally minimal to keep focus on structure and less on cosmetics.

This design system adapts Atlassian Design System, uses Tailwindcss to create css components and base styles, and references Nielsen Norman Group.

Principles

Today's web serves a wide range of audience. They may be on different devices with different operating systems. Some may use a screen reader to go along the page. Some may be less tech savvy than others. We need to encompass these considerations and form a purpose-driven solution that's simple, accessible, and functional. Grounded in this, this design system remains style-neutral and reuses what HTML has already defined as much as possible to leverage existing knowledge user has already accrued in the world wide web.

Objectives

  • Define commonly needed components
  • Define commonly needed patterns
  • Define commonly needed styles
  • Provide standards and documentation around usage
  • Provide shared vocabulary and basis to enable effective communication and efficiency across teams
