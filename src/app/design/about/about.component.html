About

Muzieh Design System is a work-in-progress design system that will allow bootstrapping new modern projects quickly. Css styles are intentionally minimalistic to keep focus on structure and less on cosmetics.

This design system adapts Atlassian Design System, uses Tailwindcss to create css components and base styles.

Principles

Today's web serves a wide range of audience. They may be on different devices and/or operating systems. Some may use a screen reader as they browse. Some may be less tech-savvy than others. For these reasons, Muzieh Design System remains design-neutral and reuses what HTML has already defined as much as possible to reduce the required learning curve.

Objectives

  • Define commonly needed components
  • Define commonly needed patterns
  • Define commonly needed styles
  • Provide standards and documentation around usage
  • Provide shared vocabulary and basis to enable effective communication and efficiency across teams
