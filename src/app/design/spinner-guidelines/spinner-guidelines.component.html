Spinner

Use a spinner to let users know content is being loaded.

Example

  • medium (default)
  • large 
                <eb-spinner size="large"></eb-spinner>
                // size options are ['medium', 'large']

Best practices

  • There should only be a single spinner on a page at one time.
  • Spinner should be placed inside container of the content being loaded.
  • Spinner should appear when user action is initiated and disappear once content loads.
