Breadcrumbs

Breadcrumbs are navigational element that convey hiearchical structure of the website. Use this to help users find find their place within a website.

Best practices

  • Place breadcrumbs horizontally before a page's main content
  • Breadcrumbs should display current page as last item, not as a link - TODO: conflicts?
  • Breadcrumbs should not be representation of session history
  • Page title and breadcrumb text should match to avoid confusion
  • Consider not using breadcrumbs if the heiarchies are flat (0-2 levels deep)

Accessibility

Should use `ol`. Should include `aria-current` and `aria-label`.
