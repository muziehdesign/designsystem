Iconography

Use icons to reduce cognitive load and allow quick identification. Check out heroicons for the complete set.

  • x-circle, solid
  • exclamation, solid
  • menu, outline
  • arrow-down, outline
  • arrow-up, outline
  • arrow-sm-up, outline
  • chevron-up, outline
  • chevron-down, outline
  • chevron-right, outline

Variants

There are two versions of icons, outline and solid. Use outline by default. Use solid version to indicate the associated action has already happened.

Best practices

  • Icons should only be used when it serves a purpose and adds value.
  • Icon should reflect the associated section or action.
  • Icon should share the same color as what they're associated with.
  • The few globally recognized icons can be used by themselves to save space. Add a `sr-only` span for screen-reader in this case. See this site's hamburger menu for example.

Accessibility

Icons are rendered with `aria-hidden=true`. `sr-only` element and associated text are relied on for accessibility support.
