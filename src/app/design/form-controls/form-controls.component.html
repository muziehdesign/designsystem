Form Controls

Input

Use input with the appropriate type value to support accessibility. Use a mask to help user with correct format when applicable.

Date input

Use a date input when collecting a date. Indicate to user what the expected format is. Always leave user the option to type in a date. It can be frustrating for users to choose dates are far away.

Same note as date input.

Select
