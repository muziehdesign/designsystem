Form Controls

Text input

Use input with the appropriate type value to support accessibility and leverage native behaviors. Choose between text, email, number, password, tel, and url. Default is text. Use a mask to help user with correct format when applicable.

Textarea

Use textarea to collect long text content from user.

Selects

Radio buttons

Radio buttons allows users to scan options quickly.

Select

A select is similar to radio button list, but allows reducing space needs. A label should be used when using a select. Default value should be populated when it's what most users will use. Group options when it helps.
