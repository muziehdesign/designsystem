Typography

Font Family

Roboto is the font used.

Font formats

Style Usage
em Use this to stress emphasis.
strong Use this to stress importance.
small Use this for secondary supporting text such as help text in form fields. It should be used sparingly.
Link Use this to take user to another location. Purpose of link can be determined from link text. This should not be used for actions.

Headings

Use headings h1 - h6 for titles and subtitles. Each page should have one h1. Heading ranks should not be skipped. All headings should reflect the document or section's content and purpose.

Main page title

Use h1 for main title of the page. Each page should have exactly one.

Key functionality headings

Use h2 to idenfiy key functionality on a page

Sub-section and field group headings

Use h3 if you have multiple logical groupings of content on the page.

Content

  • Blocks of text shoud be no more than 80 characters. See WCAG 2.1 Understanding success criterion 1.4.8: Visual Presentation for reasoning.
  • Should default to left justified, since English readers read from left to right
