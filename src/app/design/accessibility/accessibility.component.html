The 4 principles of accessibility

  • Perceivable
  • Operable
  • Understandable
  • Robust

Code

Visually hidden content

Use .sr-only for elements only meant for screen readers. Use .not-sr-only to undo.

Images

For informative images, provide concise alternative text that serves equivalent purpose of the image. Decorative images do not need alternative text. See WCAG 2.1 Guideline 1.1.1.
