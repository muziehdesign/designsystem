Tabs

Use tabs to alternate between views within the same context.

Example

  • {{ link.title }} 
                <ul ngbNav class="nav-tabs">
                    ...
                <ul>
  • {{link.link}}

Usage

  • Tabs should be parallel in nature
  • Tabs should be ordered by priority, in a way that makes sense to users
  • Tab names should be concise
  • Use sentence case for tab titles
  • See ng-bootstrap for component api and examples
