Iconography

Use icons to reduce cognitive load and allow quick identification. Check out heroicons for the complete set.

Variants

Outline

Default. Designed to be rendered at 24x24.

menu

chevron-down

Solid fill

Designed to be rendered at 20x20. Use this in buttons, form elements, to indicate action that has already taken place.

check-circle
x-circle
exclamation
eye
eye-slash

Code

                    <mz-svg-icon key="check-circle" type="outline"></mz-svg-icon>
                    // type options are ['outline', 'solid']

Best practices

  • Icons should only be used when it serves a purpose and adds value.
  • Icon should reflect the associated section or action.
  • Icon should share the same color as what they're associated with.
  • The few globally recognized icons can be used by themselves to save space. Add a `sr-only` span for screen-reader in this case. See this site's hamburger menu for example.

Accessibility

Icons are rendered with aria-hidden=true. Use an additional sr-only element with help text for accessibility support.
