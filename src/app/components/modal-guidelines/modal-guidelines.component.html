Modal

Use a modal to capture information or acknowledgement from users immediately without having users lose context.

Example

Best practices

  • Modals are invasive and should only be used sparingly
  • Modals should be used for short-term tasks
  • Avoid multiple steps inside a modal
  • Avoid modal dialogs for complex decision making
  • There should be at most one modal open at a given time

Content guidelines

Guidelines are from Atlassian's content guidelines

  • Modal dialog title should use sentence case and prioritize keywords
  • Modal dialog body should only contain valuable and relevant information that is both helpful and concise
  • Label elements with action verbs that indicate what happens when the element is selected. For example, label a select menu with Choose a user instead of Users.
