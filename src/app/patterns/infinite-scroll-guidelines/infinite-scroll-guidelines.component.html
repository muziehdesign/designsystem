Infinite scroll component

The InfiniteScrollComponent provides an efficient way to display a dynamically-loading list of items, enabling a seamless infinite scroll experience. This component supports content projection, allowing for extensive customization of item displays. It is designed to handle the dynamic loading of additional items as the user scrolls to the bottom of the container. External control over item loading and event handling for item selection is also supported.
